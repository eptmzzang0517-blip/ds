# Idea Lab

아이디어를 빠르게 검증하고 프로그램으로 구현하기 위한 실험 저장소입니다.

자동화, AI, 데이터 처리 및 생산성 향상을 위한 다양한 프로젝트를 개발합니다.

## Purpose

이 저장소의 목표는 다음과 같습니다.

- 아이디어를 빠르게 프로토타이핑
- 반복 작업 자동화
- AI 활용 가능성 검증
- 새로운 기술 실험
- 개인 생산성 향상

## Project Categories

### Automation
- 업무 자동화
- 파일 처리
- 데이터 수집
- 배치 작업

### AI
- LLM 활용
- AI Agent
- 문서 분석
- 데이터 요약
- 생성형 AI 실험

### Utilities
- 생산성 도구
- 개발 보조 도구
- CLI 도구
